# Projeyi Kullanım Adımları

projeyi klasörünü terminalde açıp `npm install` komutu ile gerekli package ları yükleyiniz.

# `npm start`

terminalde `npm start` komutu ile projeyi açınız.
Projeniz (http://localhost:3000) adresinde açılacaktır.

## Projenin kullanımı

Yeni Hesap Ekleme butonu ile açılan sayfada sosyal medya hesaplarınızı ekleyebilir ve anasayfada bulunan tabloda görüntüleyebilirsiniz. Verileriniz yerelde de tutulacağı için açma kapatma durumunda kaybolmaz.

## Projenin Yol Adımları

Projenin yapılış aşamalarını yine bu klasör içinde ProjeAdimlari adlı dosyada bulabilirsiniz.
